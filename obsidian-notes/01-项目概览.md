# 01-项目概览

## 整体架构

```
前端 (Vue 2 SPA) --HTTP/WebSocket--> 后端 (Go + Beego v2) --GORM--> 数据库
```

## 技术栈

| 层级 | 前端 | 后端 |
|------|------|------|
| 框架 | Vue 2.6.14 + Vue CLI 4.4 | Go 1.22 + Beego v2.1.0 |
| UI | Ant Design Vue 1.7.2 | - |
| 路由 | Vue Router 3.3.4 | Beego Router (180+ 路由) |
| 状态 | Vuex 3.4.0 | sync.Map + 自定义 Queue |
| ORM | - | GORM v2 |
| 图表 | ECharts 5.3.2 + Three.js 0.150.1 | - |
| 视频 | @liveqing/liveplayer | deepch/vdk + pion/webrtc |
| 通信 | Axios + WebSocket | Gorilla WebSocket + MQTT |
| 脚本 | CodeMirror 5.46 | anko |
| 加密 | crypto-js | bcrypt + SM4 + jwt-go |

## 版本信息

- **后端版本**：V3.01.RC07（2026年03月17日）
- **前端版本**：vue-antd-admin 0.7.4
- **Go 版本**：1.22.0
- **数据库**：GORM v2，支持 6 种数据库

## 核心功能模块

1. 多协议设备采集：15+ 种工业/物联网协议
2. Web 组态可视化：自研低代码编辑器，182+ 可拖拽组件
3. 实时数据推送：WebSocket 推送实时数据/告警/系统数据
4. 告警与通知：触发器引擎 + 多渠道告警（邮件/短信/钉钉/微信/语音）
5. 历史数据分析：支持 6 种历史数据库后端
6. 视频融合：RTSP/WEBRTC/GB28181/ONVIF/Monibuca
7. 脚本引擎：Anko 脚本，用户自定义逻辑
8. 组网能力：多 ISM 节点数据级联同步
9. 国产适配：达梦数据库 + 国密 SM4
