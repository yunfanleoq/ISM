# 05-数据模型

## 核心通用模型

| 模型 | 文件 | 作用 |
|------|------|------|
| DevicesModel | models/deviceLibraryModel.go | 设备模型主表，所有协议汇聚 |
| MonitorList | models/deviceLibraryModel.go | 监控树/设备实例表 |
| DeviceRealData | models/snmpDeviceModel.go | 实时数据主表 |
| DevicesAlarmList | models/deviceLibraryModel.go | 告警记录表 |
| DevicesHistoryDataList | models/deviceLibraryModel.go | 历史数据主表 |
| DevicesSupportList | models/snmpDeviceModel.go | 设备支持列表（15种协议） |

## 数据库支持

SQLite / MySQL / PostgreSQL / 达梦(DM) / SQL Server / ClickHouse

## 历史数据后端

SQLite / MySQL / PostgreSQL / InfluxDB / ClickHouse / TDengine
