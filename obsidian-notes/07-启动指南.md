# 07-启动指南

## 环境要求

### 前端

- Node.js >= 14
- npm >= 6

### 后端

- Go 1.22.0
- GCC（CGO 编译需要）

## 前端启动

```bash
cd /Users/yunfanleo/cursorProjects/ISM源码/ism-front-end
npm install
npm run serve
```

- 端口：8080
- API 代理：/api -> http://8.138.197.243:9081

## 后端启动

```bash
cd /Users/yunfanleo/cursorProjects/ISM源码/ism_server_user
go mod tidy
go run main.go
```

- 端口：8081（conf/app.conf）
- WebSocket：10215
- pprof（调试）：28081（IsDebug=true）

## 默认登录

- 用户名：admin
- 密码：admin
